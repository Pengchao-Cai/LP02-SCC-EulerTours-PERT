***** To Run *****
Using an IDE(Eclipse, IntelliJ) is recommended. 

1, create a package named "pxc190029"
2, make sure you have a package named idsa and it has Graph.java inside. 
3, put DFS.java, Euler.java and PERT.java inside. 
4, run main function of each class above.

***** Note *****
A test input of graph has already been provided in each of those main functions, but you can always input a graph from a file if you have one.
Just specify its path in the main function.
